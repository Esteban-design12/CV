Trueque U

Plataforma de intercambio para estudiantes

Trueque U es una plataforma ideada para que estudiantes universitarios puedan intercambiar, ofrecer o vender materiales, libros y recursos académicos. La web apuesta por una interfaz intuitiva, filtros por carrera/categoría, sistema de publicación de ofertas y conectividad entre usuarios.

Ver demo

Rol

Diseño UI/UX, maquetación frontend, prototipo de base de datos

Tecnologías

Wix Studio

Módulos previstos

  • Registro / Inicio de sesión de usuarios
  • Listado de publicaciones por categorías
  • Filtros por carrera, tipo de recurso, estado
  • Detalle del recurso + contacto entre usuarios

Modelo de datos & Flujo

Entidades principales

Usuarios (universitarios), Publicaciones (materiales/resources), Mensajes (interacción entre usuarios), Intercambios / Transacciones.

Interfaz clave

Home con rejilla de ofertas, filtros desplegables, página de detalle con botón de “Contactar”, panel de gestión del usuario (mis ofertas, favoritos, historial).

Diseño en rejilla

Home / Portal

Rejilla de tarjetas con miniatura, título, categoría y botón de ver más.

Detalle de recurso

Vista del recurso completo, imágenes, descripción, estado, y botón “Contactar”.

Publicar recurso

Formulario para crear nueva oferta, elegir categoría/carrera, subir imagen, definir estado.

Perfil usuario

Listado de mis ofertas, historial de intercambios, favoritos y opción de edición.
